# リストについて
`/color_lib/color_list/`に色のリストを保存すると、`color_lib.lua`から使用できるようになります。  
  
## フォーマット
CSVファイルで保存、文字コードはShift JISに設定してください。  
フォーマットは以下の通りです。  
  
`color,name,japanese,english
0xffffff,白,しろ,white`
